# SonarQube New Code Warning Fixer

## Premises

Before using this skill, ensure:

1. The SonarQube MCP server for SonarQube Server is set up for your AI agent environment (for example, see [Setup with Codex CLI](https://docs.sonarsource.com/sonarqube-mcp-server/quickstart-guide#setup-with-codex-cli)).
2. A relatively recent `python3` is installed and available on your `PATH`.
